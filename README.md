## Getting Started

### Installation

#### Using Bun (Recommended)

```bash
# Install dependencies
bun install

# Copy environment variables
cp .env.example .env
```

#### Using npm

```bash
# Install dependencies
npm install
# If you get errors try
npm install --legacy-peer-deps

# Copy environment variables
cp .env.example .env
```
